Schema Markup

Structured data is present across three confirmed page templates — homepage, collection, and contact — but contains property gaps, incorrect page types, and a missing Organization entity that affects every page on the site.

No Organization schema exists anywhere on yocrunch.com — the highest-priority fix across the entire audit. No Product schema is present on any product page. BreadcrumbList item URLs are missing sitewide, blocking breadcrumb rich result eligibility. This page documents current schema, corrected JSON-LD for each template, and a prioritised implementation sequence.

WebSite Schema

The WebSite entity is present but has an empty description field and is missing the publisher property. Fix once in the WordPress theme — all pages referencing #website inherit the correction.

Current

{
  "@type": "WebSite",
  "@id": "https://www.yocrunch.com/#website",
  "url": "https://www.yocrunch.com/",
  "name": "YoCrunch Yogurt with Toppings for all Snacking Occasions",
  "description": "",
  "potentialAction": [ { "@type": "SearchAction", ... } ],
  "inLanguage": "en-US"
}

Corrected

{
  "@type": "WebSite",
  "@id": "https://www.yocrunch.com/#website",
  "url": "https://www.yocrunch.com/",
  "name": "YoCrunch Yogurt with Toppings for all Snacking Occasions",
  "description": "YoCrunch makes lowfat yogurt with branded candy and cookie toppings including OREO®, M&M’S®, SNICKERS®, and TWIX®.",
  "publisher": { "@id": "https://www.yocrunch.com/#organization" },
  "potentialAction": [ { "@type": "SearchAction", ... } ],
  "inLanguage": "en-US"
}

Two changes: description populated with brand summary; publisher property added referencing #organization. Fix in the same theme deployment as Organization.

FIX 1 — BreadcrumbList: Add item URL to ListItem

Current

{
  "@type": "BreadcrumbList",
  "@id": "https://www.yocrunch.com/#breadcrumb",
  "itemListElement": [
    { "@type": "ListItem", "position": 1, "name": "Home" }
  ]
}

Corrected

{
  "@type": "BreadcrumbList",
  "@id": "https://www.yocrunch.com/#breadcrumb",
  "itemListElement": [
    {
      "@type": "ListItem",
      "position": 1,
      "name": "Home",
      "item": "https://www.yocrunch.com/"
    }
  ]
}

Add the item URL to the single ListItem — required for breadcrumb rich result eligibility on the homepage.

FIX 2 — CollectionPage: Add publisher property

The homepage already uses CollectionPage — no @type change needed. Add "publisher": {"@id": "https://www.yocrunch.com/#organization"} to link the page to the brand entity. Also add the same publisher to the WebSite entity (sitewide fix — see above).

Product Schema

Apply to 5 product pages

No Product schema exists on any product page. The template below is for the OREO page — apply the same structure to Cookie Dough, Snickers, M&M, and TWIX pages, updating name, description, image, and URL accordingly. Fields left blank (sku, gtin, weight, nutrition) should be added when data is available — do not fabricate values.

{
  "@context": "https://schema.org",
  "@type": "Product",
  "@id": "https://www.yocrunch.com/yogurt-with-toppings/oreo-yogurt/#product",
  "name": "OREO® Cookie Pieces with Lowfat Yogurt in 3 Flavors",
  "brand": { "@type": "Brand", "name": "YoCrunch" },
  "description": "OREO'S® with YoCrunch flavored yogurt — America's Favorite Cookie® meets your favorite yogurt.",
  "image": "https://www.yocrunch.com/wp-content/uploads/2023/10/oreo-vanilla-lowfat-yogurt.png",
  "url": "https://www.yocrunch.com/yogurt-with-toppings/oreo-yogurt/",
  "category": "Yogurt",
  "offers": {
    "@type": "Offer",
    "availability": "https://schema.org/InStoreOnly",
    "url": "https://www.yocrunch.com/yogurt-with-toppings/where-to-buy/"
  }
}

Once an AggregateRating widget is installed, add a aggregateRating property to this block. Validate each product page in the Rich Results Test before deploying.
